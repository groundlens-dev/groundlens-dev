## Hard rules

- **Never use emojis.** Use clean text and SVG assets. Badge brand logos are fine.
- **Measured, rigorous tone.** No hype. State limits plainly; never label work as "honest."
- **Verify before linking.** Confirm every image and URL resolves; a broken image is worse
  than none. Use relative paths for `assets/`, raw URLs for files in the `otwin` repo.
